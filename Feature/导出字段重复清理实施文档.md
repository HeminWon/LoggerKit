# 导出字段重复清理实施文档

## 概览

LogDetailAction.swift 中存在重复的导出相关字段，这些功能已完全迁移到 ExportFeature.swift。本文档提供清理这些重复字段的实施方案。

## 重复字段分析

### 需要移除的字段 (LogDetailAction.swift)

| 遗留字段 | 新字段 (ExportFeature) | 语义 |
|---------|---------------------|------|
| `exportLogs(format:progress:)` | `.export(.startExport(format:))` | 开始导出 |
| `exportStarted` | `.export(.exportPreparationStarted)` | 导出准备开始 |
| `exportProgressUpdated(exported:total:)` | `.export(.progressUpdated(exported:total:))` | 进度更新 |
| `exportCompleted(URL)` | `.export(.exportSucceeded(URL))` | 导出成功 |
| `exportFailed(Error)` | `.export(.exportFailed(Error))` | 导出失败 |
| `setSharePresented(Bool)` | `.export(.setShareSheetPresented(Bool))` | 分享面板显示状态 |
| `setExportErrorPresented(Bool)` | `.export(.setErrorAlertPresented(Bool))` | 错误提示显示状态 |

### 需要保留的字段

- `.export(ExportFeature.Action)` - TCA 集成点，必须保留

### 步骤 2: 迁移 LogDetailViewStore+Extensions.swift

**重构 `exportAllEventsStreaming` 方法**

当前实现 (行 469-508):
```swift

```

建议新实现:
```swift

while state.exportFeature.isExporting {

}
```
